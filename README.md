# KeyMuse Piano

A premium dark-themed web piano with recording, playback, and keyboard input. Built with vanilla HTML/CSS/JS and Tone.js.

**[Live Demo](https://piano.rohan-dhanawade.de)**

## Features

- 2-octave piano keyboard with realistic key animations
- Computer keyboard mapping (A-K keys)
- Click/touch to play with mobile support
- Recording with timing-accurate playback
- Volume control and octave selection (1-7)
- Glassmorphism dark theme with smooth animations
- How to Play guide with exercises
- Works offline, no build tools required

## Tech Stack

- [Tone.js](https://tonejs.github.io/) v15.0.4 - Audio synthesis
- [Inter](https://fonts.google.com/specimen/Inter) - Typography
- Vanilla HTML / CSS / JavaScript - No frameworks, no build step

## Development

Open `index.html` in a browser. No build step or dependencies to install.

## Deployment

Deployed to GitHub Pages via GitHub Actions. Custom domain: `piano.rohan-dhanawade.de`
